# HAPImatic Issues Archive

This directory contains an archive of all GitHub issues from the original `MattStarfield/hapimatic` repository, created on 2026-01-08 as part of the private repository migration.

## Purpose

GitHub issues cannot be transferred when creating a non-fork private clone of a repository. This archive preserves the full issue history including:

- Issue descriptions and metadata
- All comments with authors and timestamps
- Labels and state information

## Files

| File | Description |
|------|-------------|
| `issues-export.json` | Complete JSON export of all issues with full metadata |
| `issue-NNN.md` | Individual markdown files for each issue |
